# README

* Ruby version

   -  ruby '2.4.1'
   
* Environment Setup

    1 - Download Docker from https://www.docker.com/
    
    2 - cd into the folder
    
    3 - docker-compose build
    
    4 - docker-compose run app
    
    5 - rake db:create
    
    6 - rake db:migrate
    
    7 - docker-compose up
